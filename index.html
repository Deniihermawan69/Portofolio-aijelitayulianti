Proyek Saya

Membuat Aplikasi Kasir Fashion

HTML CSS PHP MySQL

Aplikasi Kasir Fashion berbasis web yang digunakan untuk mencatat dan mengelola transaksi penjualan produk fashion seperti kaos dan jaket.

Fitur Utama:

  • Input transaksi baru secara dinamis (pilih produk, jumlah, dan total harga)
  • Menyimpan dan menampilkan riwayat transaksi harian
  • Navigasi menu seperti Dashboard, Transaksi, Data Produk, Laporan, dan Logout
  • Desain antarmuka responsif dan user-friendly
  • Menggunakan database MySQL untuk menyimpan data transaksi dan produk

Proyek ini dikerjakan sebagai bagian dari tugas Ujikom dan berhasil berjalan baik di server lokal (localhost).
